`amphp/redis` is a non-blocking [`redis`](http://redis.io) client for use with the [`amp`](https://github.com/amphp/amp) concurrency framework.

**Required PHP Version**

- PHP 7

**Installation**

```bash
composer require amphp/redis dev-master
```
